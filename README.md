# wanted-pre-onboard-fe

<h2>프로젝트 실행 방법</h2>

1. git clone

```
git clone '깃허브 주소'
```

2. 버전 및 라이브러리 설치

```
npm install
```

3. 프로젝트 실행

```
npm start
```
